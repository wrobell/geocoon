GeoCoon is a library to integrate Shapely GIS geometries with Pandas data
frames.

GeoCoon supports

#. Point and line string geometries (more to come).
#. SQL/MM databases (i.e. PostgreSQL + Postgis).
#. Multiple geometry columns in a data frame.

GeoCoon requires Shapely 1.3.0, Pandas 0.13.0 and Python 3.
